#Description
This is sample app covering following functionality and use case:
- Login screen with validations where user can login with any valid email id and password
- Home screen where user will be shown three different text views.
Url- https://www.google.com is taken as base url to get the html source and then string manipulations are performed to load in three text views.
In first text view, 5th letter of response is printed In second text view, every tenth letter is concatenated and printed In third text view, total number of 'google' count is printed from response

I have also attached screenshots for your reference.

#Setup Steps
- App is created using Xcode 7 and deployment target is iOS 9
- Unzip the code, open Xcodeproj file in Xcode
- Run the code in any simulator
